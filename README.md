# DebtFreedomLab

DebtFreedomLab is a debt payoff calculator powered by AustinAI, scaling through EcoScale.  
It helps users model debt repayment strategies, visualize payoff timelines, and track progress.

---

## 🚀 Features
- Upload debt data via Excel/CSV
- Calculate payoff timelines
- Generate charts and reports
- Built with Streamlit + Python

---

## 📂 Project Structure
- `StreamlitApp.py` → main app
- `FinancialAgent.py` → payoff logic
- `DebtData.xlsx` → sample dataset
- `data/` → additional datasets
- `assets/` → logos, screenshots
- `docs/` → mission, roadmap
- `backups/` → archived versions
